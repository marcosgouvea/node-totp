# node-totp
totp with node and google authenticator

### Installation

NODE-TOTP requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and start the server.

```sh
$ cd node-totp
$ npm install 
$ node app
```

Verify the QRCode browser with Google Authenticator App

```sh
http://127.0.0.1:3000/
```
